# Documatic
Automatically generate user documentation for all your projects. Point Documatic to your repo and let it create beautiful documentation site for you.

## Motivation
Creating and maintaining user documentation is hard. I'm going to create documentation for my projects in my project repository in markdown. 

I wanted a simple way of taking my markdown documentaiton and turning it into a beautiful site. I know there are other projects that do this but they require one or more of the following:
- Running a server
- Generating static html files first
